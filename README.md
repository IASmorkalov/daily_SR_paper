# daily_SR_paper
Data and script for paper about analysing of diurnal dynamic of soil respiration with bootstrap resampling

Main data-file with field soil respiration measurements - Daily_total.xlsx

This repository contains scripts and data (both field and calculated) for the paper “On spatio-temporal upscaling of chamber-based soil respiration measurements: what matters more, how many or when?”

I would like to apologize in advance if the main script looks a bit messy: it was reconstructed from fragments that I still had after completing the analysis (the final version of the script was unfortunately lost). However, I have included all results produced by that script as well as the original input data.

The bootstrap results were larger than the 100 MB limit for uploading to GitHub, so they are available separately in cloud storage at: https://disk.yandex.ru/d/RGbBW60Saq-Ppg
